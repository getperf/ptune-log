概要
====

**ptune-log** は、日々の作業や学習を「記録し、整理し、次に活かす」ための  
個人向けワークログ支援ツールです。

Obsidian（ノートアプリ）を中心に、タスク管理・ノート記録・AI 解析を  
1 つの流れとして扱えるように設計されています。

利用には Obsidian、スマホのタイマーアプリ **ptune**、  
および Google アカウントが必要です。

本ツールの主要機能は次の 3 つです。

1. タスク支援  
2. ノートの記録  
3. 分析と振り返り（Daily Review）

以下は、これら 3 つの機能を利用した「1 日の典型的な使い方」です。


主要機能とユースケース：1日の流れ
================================

タスク支援（朝）
----------------

朝は、その日に取り組む作業を整理します。

- 今日のタスク（ToDo）の整理と登録
- Google Tasks との同期
- スマホアプリ **ptune** と連携して作業時間を記録  
  （※ ptune は作業時間を計測するタイマーアプリです）
- 整理したタスク情報は、夜の振り返りレポートにも活用されます


ノートの記録（昼）
-------------------

昼は、実際の作業内容を記録します。

- 記録用の Obsidian ノートを自動生成（すぐに記録を開始できる）
- 作業内容や学習メモをその都度追記
- AI がノートを読み取り、要点をまとめてキーワード（タグ）を抽出
- 抽出されたキーワードにより、関連するノートを後から探しやすくなる


分析と振り返り（夜）
---------------------

夜は、1 日の記録をまとめて振り返ります。

- その日に作成した記録ノートを自動で読み込み、AI が内容を整理
- よく扱ったテーマや学習内容の傾向を一覧化
- 今日うまくできたこと（Keep）を AI が抽出
- 結果を Obsidian の **Daily Note（その日のまとめノート）** に自動追記
  （※ Daily Note は Obsidian の「その日専用のメモ」です）
- 翌日の計画づくりや自己管理に活用できます


その他の機能
============

以下は必須ではありませんが、作業をさらに快適にする補助機能です。

- Obsidian の初期設定支援（推奨プラグインの案内・初期設定）
- レイアウト調整（作業しやすい画面構成への変更）
- Web ページや AI チャット履歴の保存支援
- キーワード（タグ）辞書の管理や類似タグの整理（上級者向け）


システム要件
============

- **対応OS**
  - Windows 11
  - macOS 13 以降

- **Google アカウント**
  - Google Tasks と連携してタスク管理を行います
  - 利用は無料
  - スマホタイマーアプリ **ptune**（無料）が必要です

- **AI サービス API キー**
  - ノートの要約やキーワード抽出に利用します
  - 以下のいずれかの API キーが必要です
    - OpenAI
    - Google Gemini
    - Anthropic Claude
  - いずれも有償サービスです
