セットアップ手順
================

この章では、Obsidian で **ptune-log プラグイン** を利用するための
セットアップ手順を説明します。

Windows と macOS では必要な作業が異なります：

- **Windows：タスク同期アプリ PtuneSync のインストール**
- **macOS：PtuneSync の代わりに Google OAuth クライアント設定**

.. note::

   **PtuneSync とは？（Windowsユーザー向け）**  

   PtuneSync は、Obsidian で管理しているタスクと  
   Google Tasks（スマホアプリやWebと共有されるタスク管理）を  
   自動で同期するための Windows 用アプリです。

   - 難しい Google 認証設定を自動化  
   - Obsidian と Google Tasks の間でタスクを安全に連携  
   - スマホアプリ **ptune** の作業ログと連動

   Windows では PtuneSync を入れるだけでタスク連携がそのまま使えるようになります。
