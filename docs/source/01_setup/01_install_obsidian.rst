Obsidian インストール
=======================

ここでは、Windows と macOS に Obsidian をインストールする方法を説明します。

Windows 環境のインストール
-----------------------------------

1. `公式サイト <https://obsidian.md/download>`_ にアクセス
2. Windows Installer (64bit) をダウンロード
3. ダウンロードした ``Obsidian-Setup.exe`` を実行
4. 画面の指示に従ってインストール
5. 初回起動後、Vault(ノート保存フォルダ)を作成または既存フォルダを指定

.. note::

   Windows 版は通常 ``C:\Users\<ユーザー名>\AppData\Local\Programs\Obsidian`` にインストールされます。

macOS 環境のインストール
-----------------------------------

1. `公式サイト <https://obsidian.md/download>`_ にアクセス
2. macOS 用の dmg ファイルをダウンロード
3. ダウンロードした ``Obsidian.dmg`` を開く
4. Obsidian アイコンを Applications フォルダにドラッグ＆ドロップ
5. Launchpad または Finder から Obsidian を起動
6. 初回起動後、Vault を作成または既存フォルダを指定

.. note::

   macOS で初回起動時に「開発元を検証できません」と表示される場合があります。
   その場合は、``システム設定 > プライバシーとセキュリティ`` から
   Obsidian の実行を許可してください。

Vault の作成と指定
------------------

Obsidian では、ノートは **Vault（ボルト）** と呼ばれるフォルダに保存されます。  
以降の ptune-log の利用では、この Vault がノート保存場所となります。

新しい Vault を作成する
~~~~~~~~~~~~~~~~~~~~~~~

1. Obsidian 起動後に表示される「Vault を開く」画面で **新しい Vault を作成** を選択します。
2. Vault の名前を入力します。

   - ローマ字英数字でフォルダ名を指定します。例： `MyVault`

3. 保存先フォルダを指定します。

   - Windows の例: ``C:\Users\<ユーザー名>\Documents``
   - macOS の例: ``/Users/<ユーザー名>/Documents``

既存フォルダを Vault に指定する
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. 「既存のフォルダを Vault として開く」を選択します。
2. 任意のフォルダを選択すると、その中の Markdown ファイルがノートとして扱われます。

.. note::

   Vault の実体は通常のフォルダです。  
   この中に作成する Markdown ファイルが Obsidian ノートになります。

動作確認
~~~~~~~~~~~~~~~~~~~~~~~

- Obsidian が起動し、Vault 作成または指定ができることを確認してください。
- Vault 内で新しいノートを作成し、保存されることを確認してください。
- 以降の手順でコミュニティプラグインを導入できるようになります。

.. note::

   Vault を編集する場合は画面下の以下プルダウンメニューから保管庫の管理を選択します。

コミュニティプラグイン有効化
-------------------------------------------

ptune-log を導入するには、Obsidian の設定で
**コミュニティプラグイン** を有効化する必要があります。

手順
~~~~~~~~

#. Obsidian を起動し、左下の **設定 (歯車アイコン)** をクリック
#. **General > Language** で **日本語** を選択
#. **設定 > コミュニティプラグイン** を開く
#. 「コミュニティプラグインを有効化」をクリックし、「制限モード」を **オフ** に切り替える
#. 「コミュニティプラグインを有効化しますか？」という警告が表示されたら、内容を確認して同意する

.. note::

   この設定を行うことで、公式マーケット以外のプラグインも導入可能になります。

注意点
~~~~~~~~

- **セキュリティ**  
  コミュニティプラグインは公式サポート外です。利用は自己責任となります。  

- **環境差異**  
  Windows / macOS いずれも手順は同じです。  
  画面デザインは Obsidian のバージョンによって多少異なる場合があります。

次のステップ
~~~~~~~~~~~~~

コミュニティプラグインが有効化されたら、次に **ptune-log プラグインのインストール** を行います。
